VK TG GitHub

О проекте

📚

Идея

BookReview — это платформа для читателей, где каждый может найти рецензию на любимую книгу или поделиться своим мнением с сообществом.

🛠

Технологии

Проект написан на чистом HTML и CSS без сторонних фреймворков. Всё что вы видите — результат ручной вёрстки и работы с браузерными возможностями.

🎯

Цель

Практика разработки сайта, освоение CSS-анимаций, адаптивной вёрстки и создание полноценного многостраничного сайта с нуля.

— Любимая мысль разработчика
